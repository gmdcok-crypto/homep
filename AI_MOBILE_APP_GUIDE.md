# 📱 홈페이지와 별개의 독립 모바일 앱 개발 가이드

이 문서는 기존에 만들고 있는 웹사이트(홈페이지)와는 **완전히 별개로 동작하는 정식 스마트폰 앱(Android & iOS)**을 새롭게 만들고 싶을 때, 초보자 분들이 AI(Antigravity 등)와 함께 어떻게 시작해야 하는지를 가장 쉽고 완벽하게 설명한 가이드입니다.

---

## 1. 어떤 기술로 만들어야 할까? (가장 추천하는 정석)

완전한 빈 폴더에서 새로운 앱 프로젝트를 시작할 때, 가장 추천하는 기술 조합은 **「React Native (리액트 네이티브) + Expo (엑스포)」** 입니다.

### 왜 이 조합이 최고인가요?
1. **1+1 전천후 시스템:** 코드를 딱 한 번만 짜면 **갤럭시(Android)와 아이폰(iOS)에서 모두 돌아가는 진짜 앱 2개**가 동시에 짜잔 하고 만들어집니다.
2. **배우기 쉽다:** 기존 홈페이지(웹)를 만들 때 썼던 React의 지식과 코딩 방식을 90% 이상 그대로 씁니다. 새로운 프로그래밍 언어를 또 배울 필요가 거의 없습니다.
3. **Expo(엑스포)의 마법:** 옛날에는 앱을 만들려면 수십 기가바이트(GB) 짜리 무거운 전문가용 프로그램(안드로이드 스튜디오 등)을 컴퓨터에 깔아야 했습니다. 하지만 Expo를 쓰면 그런 것 없이, 내 폰에 'Expo Go'라는 앱 하나만 깔고 컴퓨터 화면의 QR 코드만 찍으면 **내가 지금 컴퓨터로 치고 있는 코드가 내 폰 화면에 실시간으로** 바로 뜹니다.

---

## 2. AI(Antigravity)와 새 앱 프로젝트 시작하는 방법

새로운 폴더(예: `D:\my-new-app`)를 만들고 VS Code로 연 다음, AI 대화창에 이렇게 **명령(프롬프트)**을 내리시면 완벽하게 시작할 수 있습니다.

> **🗣️ 사용자 프롬프트 예시:**
> *"나는 안드로이드와 아이폰에서 동작하는 완전히 새로운 할 일 관리(ToDo) 앱을 만들고 싶어. **Expo와 React Native**를 사용해서 기본 폴더 뼈대 세팅을 진행해 주고, 제일 처음 화면 코드를 작성해 줘."*

### AI가 대신 해주는 일들
명령을 받은 AI는 여러분의 컴퓨터 터미널에서 다음 작업들을 자동으로 진행합니다.
1. `npx create-expo-app` 같은 전용 명령어를 쳐서 모바일 앱 개발용 뼈대를 세워줍니다.
2. 뼈대 세팅이 끝나면 터미널에 `npm run start` (또는 `npx expo start`)를 켜서 로컬 서버를 돌립니다.
3. 터미널 화면에 커다란 **QR 코드**가 나타나게 합니다.

👉 **여러분이 할 일:** 본인 스마트폰 카메라(또는 Expo Go 앱)로 그 QR 코드를 찍기만 하면 현재 AI가 짜고 있는 앱 화면이 내 스마트폰에 바로 실시간으로 연동되어 뜹니다!

---

## 3. 웹 개발과 모바일 앱 개발의 차이점 (주의사항)

앱은 웹사이트와 구조가 살짝 다릅니다. 이 점만 기억해 두시면 AI의 코드를 이해하기 쉽습니다.

* **화면 태그가 다름:** 웹에서는 박스를 만들 때 `<div>`나 글자를 쓸 때 `<span>` 같은 웹 전용 태그를 쓰지만, 앱에서는 앱 전용 태그인 `<View>`, `<Text>` 등을 씁니다. *(AI가 알아서 다 맞게 짜주니 크게 걱정하지 않으셔도 됩니다.)*
* **스마트폰 고유 기능:** 웹에서는 쓰기 힘들었던 스마트폰의 "카메라 켜기", "현재 내 위치(GPS) 지도에 찍기", "푸시 알람(띠링~) 울리기", "진동 징~ 울리기" 같은 고유 기능들을 코드 몇 줄로 아주 쉽게 불러다 쓸 수 있습니다.

홈페이지와 무관한 독립적인 앱을 만들고 싶으시다면, 새로운 빈 폴더를 열고 주저 없이 AI에게 **"Expo를 이용해서 React Native 앱 기초 세팅을 시작해 줘!"**라고 명령하세요. 컴퓨터 화면과 스마트폰이 마법처럼 연결되는 가장 빠르고 스마트한 모바일 앱 개발의 첫걸음이 될 것입니다.
